# MVC패턴과 스프링MVC
MVC패턴과 스프링MVC패턴을 알아보자.

## MVC패턴
Mdel + View + Controller패턴을 의미.

### 1. Model
애플리케이션의 정보. 즉 Data를 의미
### 2. View
사용자의 인터페이스. 즉 사용자가 보고 사용하는 화면을 의미
### 3. Controller
Model과 View의 중계역할. 사용자의 요청을 파악하고 그에 맞는 결과를 사용자에게 돌려줌.
