## 주요 사용 단축키

`ctr + b` 좌측 메뉴 show / close  
`ctr + space` 자동완성  
`shit + alt + f` formatter  
`ctr + shift + alt + arrow` 컬럼편집 (column select)  
`alt + arrow` 탭이동  
`ctr + shift + v` markdown preview  
`alt + 마우스drag` coloumn select  
`Ctr + T` 테마변경
