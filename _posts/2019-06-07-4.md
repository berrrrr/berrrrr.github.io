## Local에서 Debug 모드로 서버 실행 시 ‘Unable to install breakpoint ~’ 에러 발생

## 해결방법
1. Window > Preferences > Java > Compiler > Classfile Generation 항목을 모두 체크해제 > Apply > 다시 모두 체크 > Apply
2. Debug perspective로 들어가서 Remove all breakpoint
